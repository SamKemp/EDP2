Welcome to the help for the Color Matching Game

This guide will help you use and play this Color Matching Game

This is the main Menu of the game. From here you can start the game,
view the help or about pages and you can close the game.
The main Menu consists of the game's logo then four buttons, corrosponding to their actions

Here is the main game screen. From this screen the actual game can be played.
When playing the game the user looks at the color that shows up in the box and must guess what color they think it is by clicking the labeled buttons below
When the correct button is pressed the user is awarded with 10 points, if the user guesses incorrectly then it goes onto the next question.
Once 6 questions have been asked or 60 seconds have elapsed then the game ends, showing the user their final score
The final score is calculated using the number of answers the player answered correctly and if all answeres were answered correctly and time is remaining then 40 points is added to the total and one point for every second left on the clock
